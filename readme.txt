== Description ==

Mass Category Maker is designed to create as many categories as possible and easily.<br />

For people who use wordpress 3.0 or 3.0.1, please look at this ticket. http://core.trac.wordpress.org/ticket/14485.<br />
If you're planning to create NEW parent and children categories at the same time, it will happened.<br />
The temporary solution is to create another category or delete one category.<br />
Welcome to my site <a href="http://www.hotyear.com">Hot Year</a>

== Installation ==

1. Extract mass-category-maker.zip in the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Mass Category Maker" In Setting
4. Follow the on-screen instructions

== Frequently Asked Questions ==
= Children categories don't show up when I created new parent and children categories at the same time =
For people who use wordpress 3.0 or 3.0.1, please look at this ticket http://core.trac.wordpress.org/ticket/14485.
The temporary solution is to create another category or delete one category to refresh the wordpress built-in cache.
